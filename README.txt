=========================
Spionisto server software
=========================

DESCRIPTION
-----------

Spionisto is a survilleance software created from opensource technologies and 
standards such as:

 * Python
 * Grok
 * GStreamer
 * zc.buildout
 * supervisord
 * and probably others.


INSTALLATION
------------

Grab a copy of this repo and run:

    python -S bootstrap.py

After finished run:

    bin/supervisord

and open this in your browser: http://localhost:8080
